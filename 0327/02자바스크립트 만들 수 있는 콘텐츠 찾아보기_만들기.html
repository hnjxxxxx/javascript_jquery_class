  • 언론사 아이콘목록 좌,우버튼 클릭시 콘텐츠 변경
  • 오른쪽 하단 '다크모드'버튼

  • 더보기, 펼치기 버튼 클릭시 콘텐츠 추가됨

  • 좌,우 버튼

1. 자바스크립트 활동예제1 - 이미지 2장을 사용하여 버튼을 클릭하면 해당 이미지로 경로를 변경하여 전구사진 변경하기

  • onclick: 자바스크립트 이벤트의 종류 중의 하나로서 사용자가 마우스로 버튼을 클릭 하였을 경우 이벤트가 발생되어 스크립트가 동작한다.
  • onmouseover = mouseenter : 사용자가 마우스를 어떤 대상에 올렸을 경우 스크립트가 실행
  • onmouseout = mouseleave : 사용자가 마우스를 어떤 대상에 올렸다가 뺐을 경우 스크립트가 실행
  • getElementById(아이디명) : 사용자가 지정한 이름의 아이디를 선택, css의 #id선택자와 같음
  • style : 자바스크립트에서 선택한 요소에 css서식을 적용하고자 할 때 사용.
  • fontSize : 자바스크립트에서 css처럼 글자크기를 변경하고자 하는 스타일 속성.

2. 자바스크립트 활동예제2 - 버튼 클릭시 해당하는 요소를 선택하여 글자의 크기를 줄이거나/크게 변경하기

건강보험공단, 산업인력공단 등 사이트에서 사용하는 글자확대,축소하기

3. 자바스크립트 활동예제3 - footer영역에 select, option태그를 활용하여 'family site'구현하기

4. 자바스크립트 활동예제4 - 모바일 앱에서 버튼 터치시 '조명끄기', '조명켜기'기능 구현하기
